Styx Spartacus: Themes

SPARTACUS stands for Serendipity Plugin Access Repository Tool And Customization/Unification System

[S]erendipity [P]lugin [A]ccess [R]epository [T]ool [A]nd [C]ustomization/[U]nification [S]ystem

There is no own generated Styx Spartacus Themes Webpage, since Styx does not host additional themes. Please follow the provided links, which, except for the core themes, lead to S9y-Origin.
